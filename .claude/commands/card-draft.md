# /card-draft — 대충 쓴 메모를 카드뉴스용 텍스트로 정리

사용자가 대충 쓴 메모, 키워드 나열, 또는 주제만 던지면 카드뉴스에 적합한 구조화된 텍스트 파일을 생성합니다.

## 입력

- `$ARGUMENTS` : 대충 쓴 메모, 키워드, 또는 주제 (한 줄이든 여러 줄이든 OK)
- 옵션: `--file 파일명` (기본: 주제에서 자동 생성), `--tone casual|formal|fun` (기본 casual)

## 실행 절차

1. 사용자 입력을 분석하여 핵심 주제와 키워드를 추출한다
2. 아래 규칙에 따라 카드뉴스용 텍스트를 작성한다
3. `input/{파일명}.txt`로 저장한다
4. 저장 후 내용 요약을 보여주고, `/card-generate input/{파일명}.txt`로 바로 카드뉴스를 만들 수 있다고 안내한다

## 텍스트 작성 규칙

당신은 SNS 콘텐츠 전문 카피라이터입니다.

1. **제목**: `# 제목 — 부제` 형태로 시작
2. **도입부**: 독자의 공감을 끄는 1~2문장 (질문형 또는 공감형)
3. **본문 구조**: 마크다운 헤딩(`##`)으로 섹션 구분
   - 각 섹션은 카드 1~2장 분량 (3~5문장)
   - 핵심 키워드는 **볼드** 처리
   - 리스트, 표 등 시각적 구조 적극 활용
4. **마무리**: 행동 유도(CTA) 또는 여운 있는 마무리 1~2문장
5. **구분선**: 섹션 사이에 `---` 삽입

## 품질 기준

- 인스타그램 카드뉴스에 맞는 짧고 임팩트 있는 문장
- 한 문장은 30자 이내 권장
- 전문 용어는 쉽게 풀어서 설명
- 전체 흐름이 기승전결 구조
- 사용자가 쓴 핵심 내용은 반드시 포함 (임의로 빼지 않는다)
- 사용자가 쓴 톤앤매너를 최대한 살린다

## 출력

- 파일 경로: `input/{파일명}.txt`
- 파일명은 영문 snake_case (예: `coffee_guide`, `morning_routine`)
- 저장 후 아래 형태로 안내:

```
✅ input/{파일명}.txt 저장 완료

다음 명령으로 카드뉴스를 바로 만들 수 있습니다:
/card-generate input/{파일명}.txt --pages N --ratio 1:1
```
