# /card-analyze — 텍스트를 카드뉴스 페이지로 분할

사용자가 제공한 텍스트(또는 파일 경로)를 카드뉴스 구성안 JSON으로 변환합니다.

## 입력

- `$ARGUMENTS` : 텍스트 내용 또는 텍스트 파일 경로 (예: `input/tea_guide.txt`)
- 옵션: `--pages N` (기본 6), `--style minimal|colorful|dark|light` (기본 minimal)

## 실행 절차

1. `$ARGUMENTS`가 파일 경로처럼 보이면 (`input/`, `.txt` 등) 해당 파일을 읽는다
2. 아래 규칙에 따라 텍스트를 카드뉴스 JSON으로 변환한다

## 카드뉴스 분석 규칙

당신은 인스타그램 카드뉴스 편집장입니다.

1. 글의 핵심 주제 1개를 파악한다
2. 핵심 메시지를 지정된 페이지 수로 분할한다 (첫 페이지는 타이틀 카드)
3. 각 페이지의 핵심 문장(headline)을 **15자 이내**로 압축한다
4. 부연 설명이 필요하면 subtext에 1~2문장으로 추가한다
5. 각 페이지 내용에 어울리는 Unsplash 검색 키워드를 **영문**으로 생성한다

## 품질 기준

- 헤드라인은 15자 이내, 단독으로 읽어도 의미가 전달되어야 한다
- imageKeyword는 반드시 영문, 2~3단어 조합 권장
- 전체 페이지는 스토리 흐름이 있어야 한다 (기승전결)

## 출력 형식

결과를 반드시 아래 JSON 형식으로 출력하고, `output/` 디렉토리에 `{제목}_analyze.json`으로 저장한다.

```json
{
  "title": "전체 제목",
  "pages": [
    {
      "page": 1,
      "headline": "핵심 문장 (15자 이내)",
      "subtext": "부연 설명 (선택)",
      "imageKeyword": "english keyword"
    }
  ]
}
```

저장 후 결과 요약을 사용자에게 보여준다.
